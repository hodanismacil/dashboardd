# 🚀 E-Commerce Admin Dashboard

Kani waa Dashboard-ka maamulka (Admin Side) ee loogu talagalay madal e-commerce ah. Waxaa loo dhisay si casri ah oo leh naqshad nadiif ah, minimalist ah, iyo dark mode si ay u fududaato maamulista xogta ganacsiga.

---

## 🛠️ Tiknoolajiyada la Isticmaalay (Tech Stack)

### Frontend (Naqshadda)
* **React** & **TypeScript** - Habaynta component-yada iyo sugidda amniga koodhka.
* **Vite** - Qalabka dhismaha ee aadka u dheereeya.
* **Tailwind CSS** - Naqshadeynta minimalist-ka ah iyo dark mode-ka.

### Backend & Database (Xogta)
* **Node.js** & **Express** - Dhismaha server-ka iyo API-yada.
* **MongoDB Atlas** - Database-ka loogu talagalay kaydinta xogta.

---

## 🔒 Amniga & Cybersecurity Features

Mashruucan waxaa laggu dhisay xeerar adag oo dhanka digital privacy-ga iyo digital security-ga ah:
* **Rate Limiting:** Waxaa lagu daray `express-rate-limit` si IP- kasta looga xaddido weerarada (DDoS) iyo codsiyada xad-dhaafka ah ee khalad-galin kara database-ka.
* **CORS Security:** Kaliya mareegaha loo oggolaaday ayaa awooda inay la xiriiraan API-ga.

---

## 📊 Sifooyinka Ugu Waaweyn (Features)

* 📈 **GPU Market Data Charts:** Shax ku salaysan xog dhammaystiran oo si toos ah u muujinaysa qaabka suuqyada hardware-ka.
* 👥 **User Management:** Maamulista isticmaalayaasha nidaamka.
* 💬 **Customer Messages & Notifications:** Helitaanka iyo maamulista fariimaha macaamiisha si toos ah.
* 🌓 **Minimalist Dark Interface:** Naqshad indhaha u roon oo loo diyaariyay qaab professional ah.
